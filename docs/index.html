O que é o Expresso Verde?

O Expresso Verde é um sistema web desenvolvido como projeto da disciplina de Engenharia de Software da Universidade Federal do Tocantins (UFT), sob orientação do Prof. Edeilson Milhomem da Silva (Semestre 2026.1).

Nosso objetivo é aplicar conceitos rigorosos de engenharia de software na criação de uma plataforma que conecte vendedores e compradores de artigos de jardinagem de forma simples, eficiente e segura.

Funcionalidades Principais

🛒 Compra e Venda

Sistema completo para transações seguras de produtos de jardinagem.

🔍 Busca Inteligente

Encontre plantas, sementes e ferramentas com facilidade.

📱 Interface Responsiva

Acesse de qualquer dispositivo com uma experiência fluida e moderna.

🌱 Comunidade

Feed interativo para compartilhamento de dicas, textos e imagens entre os usuários.

Arquitetura e Tecnologias

O sistema utiliza o padrão arquitetural MVC (Model-View-Controller) para garantir uma base sólida e de fácil manutenção.

PHP 8.x MySQL HTML5 / CSS3 Padrão MVC GitFlow
